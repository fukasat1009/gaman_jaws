# <アプリ名>
我慢ジョーズ
## サイト概要
一つの目標（欲しい物）を設定して、日々の我慢した物を合計して成果を可視化するサイト

### サイトテーマ
「塵も積もれば山となる」の可視化

### テーマを選んだ理由
日頃節制する中で家計簿アプリなどで支出や貯蓄は記録する事ができるが、
欲しい物という目標に向かって日頃の我慢したものを記録するアプリがあれば、自分の節約の結果がわかって便利だと思ったため

### ターゲットユーザ
節約したい人、欲しいものがある人

### 主な利用シーン
金額の高い買い物を予定する際、計画をたてたい時
